# Fire Detection System using ESP32

A real-time fire detection system based on Internet of Things (IoT) using ESP32. This system monitors environmental conditions through temperature and smoke sensors, then sends alerts via Telegram when potential fire hazards are detected.

## üìñ Project Description
This project aims to detect potential fire hazards in indoor environments by monitoring temperature and smoke levels. The ESP32 processes sensor data and triggers real-time alerts when predefined thresholds are exceeded.

Notifications are delivered instantly via Telegram, enabling rapid response and improved safety monitoring.
